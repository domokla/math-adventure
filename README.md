# Math Fun (0-20)

This is a simple web game to help kids practice addition and subtraction within 0-20.

## Features
- Random addition and subtraction problems (0-20)
- Automatic feedback on answers
- Timer shows how long it takes to finish all tasks
- Reward image (dolphin or penguin) if score >= 95%

## How to use
1. Open `index.html` in your browser
2. Set the number of problems and click Start
3. Solve the problems
4. See your score and time, and enjoy the reward!
